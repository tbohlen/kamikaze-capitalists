To run Kamikaze Capitalists:

 - Make sure that you have Java 1.7 or higher installed,
   for example by running the command "java -version"

 - If you do not have Java 1.7 or higher installed,
   you can get it at http://java.com/en/download/index.jsp

 - run the Kamikaze Capitalists.jar file by double-clicking it
